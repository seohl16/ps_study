- `Python`을 이용해 일주일에 3~5문제씩 풀기

## 알고리즘 문제 풀 때!
1️⃣ 문제를 찬찬히 읽고 30분정도 고민해보기

2️⃣ 고민해본 방법으로 코드작성해보기

3️⃣ 틀리면 다시 고민해본 후 구글링해보기

4️⃣ 뭐가 다른지 생각해보기
